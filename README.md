Usage
Site visitors are first brought to the homepage where they can view the forum categories and each category's threads by clicking on that category's tab.
Visitors can search threads by name and content, and view the posts of a thread by clicking the "View Posts" link within a thread block.
Visitors can log in as a user or register a new user account by clicking "Login/Register" in the navbar.
Registering a new account requires a unique username, an email and a password.
On login the user is redirected to the previous page, and on registration the user is redirected to the homepage.
Users can view their username in the navbar, create new threads in categories, create new posts in threads, and log out.
User cookies will last for a day if they dont log out.
Admin users can delete threads, delete posts, and reach the admin page from the navbar.
In the admin page admins can view all users, their emails, admin status and registration/last login datetimes, can search users by name, and delete users.
When a user is deleted, all posts made by the user are deleted as well.
